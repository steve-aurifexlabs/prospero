ProsperoDev.Live

Collaborative Software Development

Prospero is open source software. Evaluate Prospero here with temporary collaboration sessions. Your files are deleted after 24 hours. Be sure to download your files at the end of every session.



Check your email for an authentication code.

Features

  • Video conferencing
  • Screen sharing
  • Whiteboard
  • Shared filesystem
  • Code editor
  • In-browser (No extensions/plugins/apps)
  • Open Source

Useful for

  • Pair programming
  • Developer interviews
  • Teaching coding
  • Dev + QA / UX pairing
  • Remote / hybrid teams
  • Freelance developers
  • Hackathons
